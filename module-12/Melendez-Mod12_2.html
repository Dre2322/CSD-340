5 Things I've Learned Since the Course Started

  1. Linking HTML, CSS, and External Resources: I learned how to link HTML files to CSS stylesheets and how to link to external websites, ensuring web pages are styled consistently and can connect to other resources.
  2. Box Model Concepts: I gained a strong understanding of the CSS box model, including how to manage borders, margins, padding, and control the size of elements, which is essential for effective layout and spacing on web pages.
  3. Table and Form Styling: I learned how to style tables and forms, including adding borders and backgrounds, customizing bullet points for lists, and modifying form elements to enhance their appearance and usability.
  4. Site Layout Creation: I gained knowledge of how to create entire site layouts using CSS, ensuring elements like headers, footers, sidebars, and main content areas are positioned and styled appropriately for user navigation.
  5. JavaScript Integration for Interactivity: I learned how to introduce basic JavaScript functionality, such as handling events and manipulating arrays, to add interactivity to web pages (e.g., creating a to-do list or favorite book list).
